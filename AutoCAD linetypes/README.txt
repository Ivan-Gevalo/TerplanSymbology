Файл terplan.lin предназначен для использования в системах автоматизированного проектирования (САПР) для отображения линейных объектов, границ зон и территорий в соответствии с требованиями, утвержденными приказом Министерства экономического развития Российской Федерации от 09.01.2018 № 10.

ВАЖНО: для корректного отображения отдельных типов линий необходимо выполнить следующие действия:
1) установить в систему шрифт TerplanSymbology.ttf;

2) в диспетчере стилей текста в текущем чертеже необходимо настроить стиль текста Terplan и установить для данного стиля шрифт TerplanSymbology.

Файл чертежа terplan_linetypes.dwg сохранен в формате AutoCAD DWG 2010 и содержит образцы типов линий, описанных в файле terplan.lin.
